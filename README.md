# 📺 FreeZap — Android wrapper

Capacitor wrapper that packages the [freezap](https://github.com/abourdim/freezap) web app as an installable **Android APK**.

> فريزاب · FreeZap

A web-based remote for the Freebox Player. Pilot your Freebox Revolution from any phone, tablet, or laptop on the same WiFi. No app store, no install, no account. Trilingual FR/EN/AR, 8 themes, keyboard shortcuts.

## Quick start

```bash
./manage.sh    # auto-detects your OS, then opens the menu
```

Build, test on emulator, and prepare a Play Store release from a single guided menu. See [WORKSHOP.html](WORKSHOP.html) for the full step-by-step guide.

## Download the latest APK

[📦 v1.0-debug release](https://github.com/abourdim/freezap-android/releases/latest) · [![QR](releases/download-qr.svg)](https://github.com/abourdim/freezap-android/releases/latest)

Scan the QR with an Android phone, or `adb install -r freezap-v1.0-debug.apk`.

## Capabilities required at runtime

- Standard WebView (no special hardware)
